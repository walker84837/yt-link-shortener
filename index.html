YouTube Link Shortener

Simply enter the YouTube link and click the 'shorten' button to shorten the link.

This site works with everything client-side, so you can even access the JavaScript file from your browser. If you'd like to contribute to this project, please do so in my repository.

The site is designed with a clean and modern aesthetic. It uses the Open Sans font for visually appealing typography. The background colour is a deep red tone, creating a visually striking contrast with the text colour.

This site provides a convenient way to shorten and clean up YouTube links, giving users a streamlined experience for sharing and embedding YouTube videos.
